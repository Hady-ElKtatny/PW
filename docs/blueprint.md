# **App Name**: Competency Bridge Portfolio

## Core Features:

- Hero Section Display: Displays the main headline, tagline, and a call-to-action button.
- Journey Visualization: Presents the entrepreneurial journey through text and key statistics cards about the competency gap. This feature also uses data cards to display graduate unemployment rates, curriculum relevance, and the challenges of entry-level jobs.
- Solution Presentation: Highlights the clinical residency solution with a feature grid. This feature focuses on the core concepts such as 'Simulated Work', 'Verified Portfolio', and the 'USP' that turns zero years of experience into one year of verified competency.
- Milestones Timeline: Displays a timeline visualizing the project milestones, including research & validation, identifying the blame game, and developing the fellowship model. Also presents placeholders for diagrams using images with alt text for 'Lean Canvas Diagram' and 'Skills Canyon Diagram'.
- Future Vision Display: Outlines the road ahead and the north star metric which is the placement rate within 90 days of graduation. It sets the vision to dismantle the 3-Year Wall.
- Contact Section Navigation: Enables users to contact by showing an email, and quick-links to various online profiles like LinkedIn or Github.

## Style Guidelines:

- Primary color: Deep Blue (#293B5F) to convey trust and corporate professionalism, aligning with the brand's message.
- Background color: Light Gray (#F0F0F0), almost white, for a clean and academic look.
- Accent color: Pale Blue (#77ACF1), subtly lighter than the primary color, used to highlight key elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif. Suitable for both headlines and body text.
- Responsive single-page layout using Flexbox and Grid for modern design and adaptability across devices.
- Use minimalist, professional icons to represent key concepts. Ensure they are easily recognizable and complement the academic aesthetic.
- Subtle transition animations for smooth scrolling between sections.
